# BIOS2000 for Laravel

## Requirements

https://github.com/Microsoft/msphpsql/

## Installation

```
composer require bernhardk/laravel-bios2000
```

In ".env" the following needs to be added and configured:
```
BIOS_CONNECTION=mssql
BIOS_HOST=...
BIOS_PORT=1433
BIOS_USERNAME=...
BIOS_PASSWORD=...
BIOS_DB_S=...
BIOS_DB_M=...
BIOS_DB_A=...
```
